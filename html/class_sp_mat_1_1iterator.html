Detailed Description

template<typename eT>
class SpMat< eT >::iterator

So that we can iterate over nonzero values, we need an iterator implementation. This can't be as simple as Mat's, which is just a pointer to an eT. If a value is set to 0 using this iterator, the iterator is no longer valid!
