doublecomplex* oGetDoubleComplexFromPointer ( double *  _pdblReal,
double *  _pdblImg,
int  _iSize 
)

Returns a target doublecomplex array constructed from the source real and imaginary parts. The real and imaginary parts can be NULL or not NULL :

  • if real part and imaginary part of source array are not NULL, the returned array is as expected,
  • if real part of source array is NULL and imaginary part is not NULL, the real part of the returned array is filled with zeros,
  • if real part of source array is not NULL and imaginary part is NULL, the imaginary part of the returned array is filled with zeros,
  • if both real and imaginary parts of source array are NULL, the returned array is NULL.
    Parameters
    _pdblRealthe real part of the source array
    _pdblImgthe imaginary part of the source array
    _iSizethe size of the source array
    Returns
    a target doublecomplex array constructed from the source real and imaginary parts.
