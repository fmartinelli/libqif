Detailed Description

This is a generic engine plotter class.

This class is a generic plotter that uses the Scilab_plotter,GNU_plotter,Matlab_plotter and Maple_plotter classes under the under the user's choice.

unsigned short int Plotter::plotter [protected]

This integer will be used as a flag to determine with which plotter draw. by default will be -1.
0 : SciLab
1 : GNU-Plot
2 : MatLab
3 : Maple
