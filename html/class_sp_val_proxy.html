Detailed Description

template<typename eT>
class SpValProxy< eT >

Sparse value proxy class, meant to prevent 0s from being added to sparse matrices. T1 should be either SpMat or SpSubview, and if it's not, bad news is probably coming. This class only uses T1::add_element() and T1::delete_element().
