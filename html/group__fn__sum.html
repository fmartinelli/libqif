Delayed sum of elements of a matrix along a specified dimension (either rows or columns). The result is stored in a dense matrix that has either one column or one row. For dim = 0, find the sum of each column (traverse across rows) For dim = 1, find the sum of each row (traverse across columns) The default is dim = 0. NOTE: the dim argument is different than in Matlab/Octave.
